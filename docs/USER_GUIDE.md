## Navigation Overview
After signing in, you are taken to the **Homepage** — a grid of feature cards giving direct access to every module:

| Card | What it does |
|---|---|
| **Dashboard** | Overview of school metrics, charts, recent activity |
| **Students** | Student records, admissions, attendance, status |
| **Academics** | Examinations, scores, reports, question bank |
| **Fees Management** | Fees, payments, bills, debtors, financial profiles |
| **Canteen** | Canteen payments, calendar, debtors |
| **Staff Management** | Staff records, attendance, leave, performance |
| **Payroll** | Salaries, payslips, SSNIT/PAYE schedules |
| **Finance** | Income, expenses, budgets, financial reports |
| **Notifications** | SMS, Email, WhatsApp messaging |
| **Settings** | School identity, users, permissions, branding |

Every module page also has a left sidebar for quick switching between modules.

## Roles & Permissions
Nickland Edusoft uses a role-based access control system with 10 default roles: Proprietor, Administrator, Head Teacher, Class Teacher, Subject Teacher, Accountant, Secretary, Cook, Security, Cleaner.

Each role has default permissions that can be customised in **Settings → Designations**. Individual users can also be given extra access via permission overrides in **Settings → Users & Access**.
