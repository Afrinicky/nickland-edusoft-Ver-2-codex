**Copyright © 2026 Nickland Sales. All rights reserved.**

## System Requirements
- Windows 10 or Windows 11 (64-bit)
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space
- 1366×768 display minimum (1920×1080 recommended)

## Installing the Software

1. Run `Nickland Edusoft Setup 2.0.0.exe`
2. Choose installation directory (default: `C:\Program Files\Nickland Edusoft\`)
3. Select desktop and Start Menu shortcut options
4. Click **Install** and wait for completion
5. Launch from the desktop shortcut or Start Menu

## First Launch — Administrator Bootstrap

On the very first launch, the **Create Administrator Account** screen appears:
1. Enter the Administrator's full name
2. Choose a username (e.g., `admin`)
3. Set a password (minimum 6 characters)
4. Confirm the password
5. Click **Create Account & Continue**

This screen only appears once. From the next launch onwards, the standard login screen is shown.

## Setting Up Your School

After signing in for the first time, go to **Settings → School Identity** and fill in:
- School name, motto, vision, mission
- Address and contact details
- Registration numbers (GES, TIN, SSNIT)
- Upload school logo (PNG/JPEG, recommended 200×200px)
- Theme colors (defaults: navy `#1B3A6B` + gold `#C9961A`)

Then create user accounts for staff in **Settings → Users & Access**.

## Data Location
All school data is stored locally at:
```
%APPDATA%\NicklandEdusoft\nickland-edusoft.db
```
Photos and documents are stored at:
```
%APPDATA%\NicklandEdusoft\uploads\
```

## Backup
Use **Settings → Backup & Sync** to:
- Take manual local backups
- Connect Google Drive for automated cloud backup
